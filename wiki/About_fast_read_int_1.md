# About fast read int 1

一个有趣的事是“读取字符的`getchar()`比`scanf`和`cin`都要快”

基于此，可以使用`getchar()`来读取每一位数字，然后合成为一个数，大幅提升读取效率

因而可以考虑一个数$\overline{abc}$，用`getchar()`读入$a$后，将其乘$10$再与`getchar()`读入的$b$相加。重复此操作只到读入所有的数字并合成为一个数

一些性能的比较：

| 方法  | 耗时  | 备注      |
| ----- | ----- | --------- |
| scanf | 880ms |           |
| cin   | 640ms | 取消 sync |
| fri1  | 228ms |
